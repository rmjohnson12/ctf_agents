# Forensics Agent

Specialist agent for digital forensics challenges.

## Capabilities
- Memory forensics
- Disk forensics
- Network packet analysis
- Steganography detection
- File carving
- Metadata analysis

## Tools Used
- Volatility (memory analysis)
- Binwalk, Foremost (file carving)
- Wireshark, tshark (network analysis)
- ExifTool (metadata)
- Steghide, zsteg (steganography)
- Autopsy, FTK

## Common Techniques
- Memory dump analysis
- Deleted file recovery
- PCAP analysis
- Hidden data extraction
- Timeline analysis
