# Reverse Engineering Agent

Specialist agent for reverse engineering challenges.

## Capabilities
- Binary analysis
- Disassembly and decompilation
- Debugging and dynamic analysis
- Obfuscation removal
- Malware analysis
- String and function extraction

## Tools Used
- Ghidra, IDA Pro
- Radare2
- GDB with plugins (PEDA, pwndbg)
- objdump, strings
- Binary Ninja

## Common Techniques
- Static analysis
- Dynamic analysis with debuggers
- Control flow analysis
- Data flow analysis
- Anti-debugging detection and bypass
